# 涨单宝

## 项目介绍

涨单宝基于 `uni-app` 框架，使用 `Vue3` + `TypeScript` + `Sass` +`Vite` + `Wot Design Uni` + `UnoCSS` 开发

## 技术栈

`uni-app` + `Vue3` + `TypeScript` +`Sass` + `Vite` + `Wot Design Uni` + `UnoCSS`

## 安装教程

1. 安装依赖

   ```bash
   npm install
   ```

2. 运行项目

   通过 `HBuildX` 运行项目

3. 打包项目

   通过 `HBuildX` 打包项目
